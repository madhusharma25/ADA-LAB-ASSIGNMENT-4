This project is part of Analysis and Design of Algorithms (ADA) Lab Assignment – 4. It focuses on advanced algorithm techniques and their performance analysis using Python.

The assignment includes:

Task 1: Backtracking algorithm for flight crew scheduling.
Task 2: Branch and Bound approach for optimization.
Task 3: String matching using Naive and KMP algorithms.
Task 4: Performance comparison of string matching algorithms with graphs.
Task 5: Conceptual understanding of backtracking, NP-hard problems, and heuristics.

Graphs and outputs are used to analyze efficiency and compare algorithm performance.
